Dummy Text
Use the attribute "data-dummy" for dummy text
<p data-dummy></p>

Specify a number of dummy words
<em data-dummy="20"></em>
Random number of dummy words
<div data-dummy="3,6"></div>
Dummy Images
Use the attribute "data-dummy" for a dummy image
Autosize to parent container
<img data-dummy />
Specific Size
<img data-dummy="400x300" />
or,
<img data-dummy width="400" height="300" />
or, you can also set the size with CSS
Custom Text
<img data-dummy data-text="User Avatar" />
Custom Colors
<img data-dummy data-color="#0000ff" data-text-color="#fff" />
Repeatable Elements
Repeat an element `data-repeat="3"`
<ul> <li data-repeat="3">My list item</li> </ul>
  • My list item
Combine with `data-dummy`
<ul> <li data-dummy="3" data-repeat="3"></li> </ul>
Random Range `data-repeat="2,5"`
<ul> <li data-dummy="1,3" data-repeat="2,5"></li> </ul>

Backstory

In a continuous effort to streamline my front-end build process, I was finding that testing images and text was a time consuming step, but also a crucial one in simulating dynamic content, testing word wrapping, repeating of elements, etc. To battle test these edge cases not only quicker, but more thoroughly, the first version of dummy.js was realised.

Packaging up DummyJs was both a step in providing quicker access to the script and opening it up for everyone to use – community ♥.

Hopefully DummyJs can plug into your workflow and help crash test your next build!
